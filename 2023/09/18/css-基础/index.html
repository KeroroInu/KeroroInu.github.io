CSS 基础知识点

1.css 基础

1.1 基础选择器

  • 标签选择器
  • 类选择器
  • id 选择器
  • 通配符选择器

1.2 选择器进阶

  • 后代选择器：空格
  • 子代选择器：>
  • 并集选择器：，
  • 交集选择器：紧挨着
  • hover 伪类选择器

1.3 结构伪类选择器

  • E:first-child{}
  • E:last-child{}
  • E:nth-child(n){}
  • E:nth-last-child(n){}
    • n的注意点：
      • n 为：0、1、2……
      • 可以通过 n 来组成公式
功能 公式
偶数 2n、even
奇数 2n+1、2n-1、odd
找到前 5 个 -n+5
找到从第5 个往后 n+5

1.4 伪元素

  • 伪元素：一般页面中的非主题内容可以使用为元素
    • 区别
      • 元素：HTML 设置的元素
      • 伪元素：由 CSS 模拟出来的标签效果
    • :: before （在父元素内容的最前添加一个伪元素）
    • :: after （在父元素内容的最后添加一个伪元素）
  • 注意点：
    • 必须设置 content 属性才能生效
    • 伪元素默认是行内元素

1.5 行高

  • 属性名：line-height
  • 取值：
    • 数字 + px
    • 倍数（当前标签 font-size 的倍数）
  • 应用：
    • 让单行文本垂直居中可以设置 line-height: 文字父元素高度
    • 页面精确布局时，会设置 line-height: 1可以取消上下间距
  • 行高与 font 连写的注意点：
    • 如果同时设置了行高和 font 连写，注意覆盖问题
    • font: style weight size/line-height familt;

1.6 标签水平居中方法

  • margin: 0 auto
    • 一般针对于固定宽度的盒子，如果大盒子没有设置宽度，此时会默认占满父元素的宽度

margin: 0 auto; 标签居中

Text-align: center; 内容居中

1.7 元素显示模式

1.7.1 块级元素

  • 显示特点
    • 独占一行（一行只能显示一个）
    • 宽度默认是父元素的宽度，高度默认由内容撑开
    • 可以设置宽高
  • 代表标签：
    • div 、 p 、 h 系列
    • ul、li、ol、dl、dt、dd、form、header、nav、footer……

1.7.2 行内元素

  • 显示特点
    • 一行可以显示多个
    • 宽度和高度默认由内容撑开
    • 不可以设置宽高
  • 代表标签
    • a 、 span
    • b、u、s、i、strong、ins、em、del

1.7.3 行内块元素

  • 显示特点：
    • 一行可以显示多个
    • 可以设置宽高
  • 代表标签：
    • input、textarea、button、select……
    • 特殊情况
      • img 标签有行内块元素特点，但是 Chrome 浏览器中显示结果为 inline

1.7.4 HTML嵌套注意点

  1. 块级元素一般作为大容器，可以嵌套：文本、块级元素、行内元素、行内块元素等等……
    • 但是：p 标签中不要嵌套 div、p、h等块级元素
  2. a 标签内部可以嵌套任意元素
    • 但是：a 标签不能嵌套 a 标签

2.css 优先级

2.1 优先级公式

  • 优先级公式：（从小到大）
    • 继承
    • 通配符选择器
    • 标签选择器
    • 类选择器
    • id 选择器
    • 行内样式
    • !important
  • 注意点：
    • !important写在属性值的后面，分号的前面！
    • !important不能提升继承的优先级，只要是继承，优先级最低！
    • 实际开发中不建议使用!important

2.2 权重叠加计算

  • 权重叠加计算公式：（每一级之间不存在进位）
    • 符合选择器：
    • （行内样式个数，id选择器个数，类选择器个数，标签选择器个数）
  • 比较规则：
    • 先比较第一级数字，如果比较出来了，之后不用看了
    • 若第一级相同，再去看看第二级数字，以此类推
    • 最终所有数字相同，表示优先级相同，则比较层叠性（谁在下面，谁说了算）
    • 若!important不是继承，则权重最高！

3.3 外边距折叠现象

3.3.1 合并现象

垂直布局的两个块级元素，上下 margin 会合并，两者间的距离为 margin 的最大值

  • 解决方法：避免就好
    • 给其中一个盒子设置margin 即可

3.3.2 坍塌现象

互相嵌套的块级元素，子元素的 margin-top 会作用在父元素上，导致父元素一起往下移动

解决方法：

  1. 给父元素设置 border-top 或者 padding-top（分隔父子元素的 margin-top）
  2. 给父元素设置 overflow：hidden
  3. 转换成行内块元素
  4. 设置浮动

4.浮动

4.1 特点

  1. 浮动元素会脱标，在标准流中不占位置（相当于从地面飘到空中）
  2. 浮动元素比标准流高个级别，可以覆盖标准流中的元素
  3. 浮动找浮动，下一个浮动元素会在上一个浮动元素左右浮动
  4. 浮动元素有特殊的显示效果
    1. 一行可以显示多个
    2. 可以设置宽高
    3. 相当于具备了行内块标签的特点
    4. 顶对齐
  5. 注意点：
    1. 浮动的元素不能通过 text-align:center居中
    2. 通过父元素 margin: 0 auto 居中

4.2 css 书写顺序

  • 可以让浏览器执行效率更高
    1. 浮动/display
    2. 盒子模型：margin border pedding 宽度 高度 背景色
    3. 文字样式

4.3 清除浮动

4.3.1 清除浮动介绍

  • 含义：清除浮动带来的影响
    • 影响：如果子元素浮动了，此时子元素不能撑开标准流的块级父元素
  • 原因：
    • 脱标后，子元素不占位置
  • 清除的目的：父元素需要高度，从而不影响页面布局

4.3.2 清除浮动的方法

  1. 给父元素设置高度
  2. 额外标签法
  3. 单伪元素清除法
  1. 双伪元素清除法
  1. 给父元素设置 overflow：hidden

6.定位修饰

6.1 静态定位

  • static 默认添加，即没有定位

6.2 相对定位

  • 介绍：相对型定位，相对于自己之前的位置进行移动
  • 代码：position：relative
  • 特点：
    1. 需要配合方位属性实现移动
    2. 相对于自己原来的位置进行移动
    3. 在页面位置中占位子 ，没有脱标
  • 应用场景：
    1. 配合绝对定位，组成 cp（子绝父相）
    2. 用于小范围的移动

如果 left 和 right 都有，以 left 为准

如果 top 和 bottom 都有，以 top 为准

6.3 绝对定位

  • 介绍：拼爹型定位，相对于非静态定位的父元素进行定位移动
  • 代码：position：absoulte
  • 特点：
    1. 需要配合方位属性实现移动
    2. 默认相对浏览器可视区域进行移动
    3. 在网页中不占位置，脱标
  • 应用场景：
    • 配合相对定位

6.3.1 居中

6.3.2 位移居中

6.4 固定定位

  • 介绍：死心眼型定位，相对于浏览器进行定位移动
  • 代码：position：fixed
  • 特点
    1. 需要配合方位属性实现移动
    2. 相对于浏览器可视区域进行移动
    3. 在页面中不占位置，脱标
  • 应用场景：
    • 让盒子固定在屏幕的某个位置

6.5 显示层级

  • 不同布局元素的层级关系
    • 标准流 < 浮动 < 定位
  • 不同定位之间的层级关系：
    • 相对、绝对、固定默认层级相同
    • 此时 HTML 中写在下面的元素层级更高，会覆盖上面的元素

7. 装饰

7.1 基线

  • 基线： 浏览器文本类型元素排版中存在用于对齐的基线（baseline）

7.2 垂直对齐方式

  • 属性名：vertical-align
  • 属性值：
    • baseline 基线对齐
    • top 顶部对齐
    • middle 中部对齐
    • bottom 底部对齐

7.3 光标类型

  • 场景：设置鼠标光标在元素上时显示的样式
  • 属性名: cursor
  • 属性值：
    • default
    • pointer
    • text
    • move

7.4 边框圆角

  • 属性名：border-radius

  • 赋值规则：从左上角开始赋值，然后顺时针赋值，没有赋值的看对脚

  • 正圆：

    • 盒子必须是正方形

    • 设置边框圆角为盒子宽高的一半

      • border-radius: 50%

      • border-radius: 宽高一半 px

  • 胶囊按钮：

    • 盒子要求是长方形
    • 设置
      • border-radius：盒子高度的一半

7.5 overflow 溢出部分显示效果

  • 溢出部分：指的是盒子 内容部分 所超出盒子范围的区域
  • 场景： 控制内容溢出部分的显示效果，如：显示、隐藏、滚动条……
  • 属性名：overflow
    • visible 默认值，溢出部分课件
    • hidden 溢出部分隐藏
    • scroll 无论是否溢出，都显示滚动条
    • auto 根据是否溢出，自动显示或隐藏滚动条

7.6 元素本身隐藏

  • 常见属性：
    • visbility：hidden；
    • display：none；
  • 区别：
    • visibility：是占位隐藏
    • display：是不占位隐藏

7.7 元素整体透明度

  • 场景：让某元素整体（包括内容）一起变透明
  • 属性名：opcity
  • 属性值：0~1
    • 1：表示完全不透明
    • 0：表示完全透明

8.补充样式

8.1 精灵图

  • 优点：减少服务器发送次数，减轻服务器的压力，提高网页加载速度

  • 使用：

    • 创建一个盒子，设置盒子的尺寸和小图尺寸相同
    • 将精灵图设置为盒子的背景图片
    • 修改背景图片位置
      • 通过 PxCook 测量小图片左上角左边，分别取负值设置
      • 给盒子的 background-position: x y;

8.2 背景图片大小

  • 语法：background-size：宽度 高度；
  • 取值
    • 数字 + px
    • 百分比
    • contain 包含
    • cover 覆盖

8.3 阴影

  • text-shadow
  • box-shadow

必须：

​ 水平偏移量：h-shadow

​ 垂直偏移量：v-shadow

8.4 过渡

  • 属性名：transition
  • 常见取值：
    • 过度的属性：all 或 width
    • 过度的时长：数字 + s
