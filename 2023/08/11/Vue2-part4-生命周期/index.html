Vue2-part4-生命周期

一、Vue生命周期

Vue 生命周期：就是一个 Vue 实例从创建到销毁的整个过程

生命周期的四个阶段：① 创建 ② 挂载 ③ 更新 ④ 销毁

  1. 创建阶段：创建响应式数据
  2. 挂载阶段：渲染模版
  3. 更新阶段：修改数据，更新视图
  4. 销毁阶段：销毁 Vue 实例

二、Vue 生命周期钩子

Vue 生命周期过程中，会自动运行一些函数，被称为【生命周期钩子】（hook，译为🪝） => 让开发者可以在【特定阶段】运行自己的代码
